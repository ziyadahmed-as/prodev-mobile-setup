# Mobile Development Setup (Expo Go)

## Objective
Set up a React Native development environment using Expo and test apps on a physical device using Expo Go.

## Prerequisites
- Node.js LTS installed
- VS Code installed (recommended)
- Expo Go on your Android/iOS device
- Computer and device on same Wi-Fi (for LAN QR scanning)

## Steps performed
1. Verified Node.js & npm:
   - `node -v` → v18.x / v20.x
   - `npm -v` → x.x.x

2. Created Expo app:
   ```bash
   npx create-expo-app prodev-test
   cd prodev-test
